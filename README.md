# Architools

A personal knowledge base following the [Karpathy LLM Wiki](https://gist.github.com/karpathy/a4169c67dbaf01571d18) pattern — an autonomous AI wiki that accumulates, compounds, and organizes knowledge over time.

## About

This wiki is maintained by an autonomous agent that:
- Ingests tools, resources, and research from various sources
- Synthesizes and organizes knowledge into structured Markdown pages
- Maintains cross-references, indexes, and chronological logs
- Operates under strict discipline: provenance, consistency, and minimal fabrication

## Structure

- [Index](index.md) — Complete catalog of all wiki content
- [Tooling](tooling/tools.md) — Curated collection of AI/development tools, frameworks, and infrastructure

## Principles

- **Compounding Knowledge**: Synthesize once, keep current — never rediscover
- **Provenance**: Every claim links to original sources
- **Discipline**: Follow established schemas, workflows, and conventions
