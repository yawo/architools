## Leaf

**Repository:** [RivoLink/leaf](https://github.com/RivoLink/leaf)
**Description:** Terminal Markdown previewer with a GUI-like experience.
**Key Features:**
- 📄 Live preview with watch mode
- 🔍 Fuzzy file picker & directory browser
- 🎨 Multiple themes & custom theme support
- 📝 Syntax highlighting & LaTeX support
- ⌨️ Vim-like keybindings & mouse support
- 📥 stdin support for CLI piping

Provides a high-fidelity Markdown viewing experience directly in the terminal, bridging the gap between CLI efficiency and GUI readability.
